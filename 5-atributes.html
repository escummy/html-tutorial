Existen 2 tipos en términos generales: Comunes: Su sintaxis es --> atributo = "valor" Booleanos: Su sintaxis es simplemente --> atributo (true/false) class -> Éste atributo se usa para dar estilo es CSS, por lo que se recomienda no usarlo en aqui en el HTML id -> Es un identificador único que se utiliza para seleccionar un elemento desde JavaScript y tambien para hacer navegación a través de anclas title -> Es un atributo que ayuda a la accesibilidad, mostrando una descripción del elemento al que pertenece (al pasar el cursor aparece el mensaje en un Tooltip) data -> Es un atributo muy útil que nos permite guardar un valor en la etiqueta html y cuando usemos JavaScript nos va ser necesario
